I was part of the avionics team in the Teknofest Rocket Competition. I programmed pressure, temperature, and communication sensors using Arduino. During my internship, I developed RESTful APIs using Spring Boot and ensured their performance through comprehensive testing. This internship gave me strong full-stack development experience.

Recipe Finder Application

I developed a recipe book application using C++ and Qt. The application offers various recipes based on the ingredients selected by the user. With a user-friendly interface, users can enter ingredients, and the application quickly suggests recipes that match the inputs. This project improved my desktop application development and UI design skills.

Path Blocker Game

I designed and developed the Path Blocker Game using Java and image processing techniques. The game challenges players to find the shortest path from start to finish while avoiding obstacles. It uses the Breadth-First Search (BFS) algorithm for pathfinding and integrates dynamic visual elements to enhance the gameplay experience.
